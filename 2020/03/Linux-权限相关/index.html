文件类型和权限
drwxr-xr-x

第一位 d 表明档案类型是 目录

字符 含义
d 目录
- 文件
| 链接文件
b 设备文件中可供存储的周边设备, 可随机读取设备
c 序列号设备, 如键盘鼠标, 一次性读取设备

后面每三位一组 为rwx的组合 分别代表 读, 写, 执行
三组 第一组文件拥有者权限 第二组群组中账号权限 第三组非本人且没有加入本群组的其他账号权限

第三部分表示这个文件的拥有者账号
第四部分表示这个文件所属群组

相关命令
-chgrp 改变文件所属群组
-chown 改变文件拥有者
-chmod 改变文件权限
