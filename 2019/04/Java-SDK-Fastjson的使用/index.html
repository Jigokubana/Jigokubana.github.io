    1. 您首先需要设置除了showapi_sign之外的所有必传参数，例如：
      ?title=足球&page=1&pag=for_test&showapi_appid=123

    2. 对上述参数key进行排序按照字典序(a-z)，请注意byte[]类型的参数不参与排序和计算签名，比如上传的文件；空值的参数也不参与排序和计算签名。排序后以key+value方式拼装字符串如下：
