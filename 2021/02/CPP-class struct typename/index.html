CPP null 2021/02/24

class 和 struct的区别

  1. class默认访问权限是private, struct默认访问权限是public
  2. class默认private继承, struct默认public继承
  3. struct在没有构造函数, 可以使用大括号初始化. class则要求全部public下才能大括号初始化
